### 3-2. 핵심 기능 개발 (W7~W13 / 7주: 07.24~09.16)

개발은 **하위 레이어(L1~L3) → 상위 레이어(L4~L6)** 순으로 진행하며, 일부 모듈은 병렬 개발한다.

| 주차 | 날짜 | 개발 모듈 | 주요 기능 | 산출물 |
|------|------|----------|----------|--------|
| **W7**<br>07.24~07.30 | **L1~L3 인프라 + SRV-AUTH** | - L1~L3: 공통 Entity, RepositoryContract, Domain Service<br>- **SRV-AUTH** | - User / RoleAssignment 도메인<br>- 회원가입 · 로그인 · RBAC 가드<br>- Passport OTP(기관코드 인증)<br>- DB 마이그레이션 v1.0 | AUTH API, Unit Test, Auth PR |
| **W8**<br>07.31~08.06 | **SRV-FAMILY + SRV-TEMPLATE** | - **SRV-FAMILY**: 당사자 프로파일<br>- **SRV-TEMPLATE**: 템플릿 엔진 | - 당사자 CRUD · 생애주기 자동 계산<br>- 템플릿 정의 · 10필드타입 렌더러<br>- 템플릿 × 생애주기 × 카테고리 매핑 | FAMILY API, TEMPLATE API, Template Editor PR |
| **W9**<br>08.07~08.13 | **SRV-RECORD (Core)** | - **SRV-RECORD**: 기록 CRUD | - 템플릿 기반 기록 생성·수정·삭제<br>- 자동 임시저장(5분 interval)<br>- JSONB 저장 · GIN 인덱싱<br>- 파일 첨부(20MB, EXIF 제거) | RECORD API v1, File Upload PR |
| **W10**<br>08.14~08.20 | **SRV-RECORD (Query) + SRV-WORK** | - **SRV-RECORD**: 조회/검색<br>- **SRV-WORK**: 초대/권한 | - 타임라인 뷰(cursor 기반 무한 스크롤)<br>- 기록 전문 검색(JSONB GIN)<br>- 기록자 초대(이메일, 역할·기간 지정)<br>- 권한 회수 · 공개범위 제어 | RECORD Query PR, WORK API v1 |
| **W11**<br>08.21~08.27 | **SRV-WORK(권한이양) + SRV-NOTIFY** | - **SRV-WORK**: 권한 이양<br>- **SRV-NOTIFY** | - 만 18세 자동 감지(Cron)<br>- D-7 경고 · 상태 머신(Locked→Unlocked)<br>- 동의서 전자 서명<br>- 앱 내/이메일 알림 발송 | TRANSITION API, NOTIFY API |
| **W12**<br>08.28~09.03 | **SRV-SHARE + SRV-PROFILE** | - **SRV-SHARE**: PDF/공유<br>- **SRV-PROFILE**: 당사자/전문가 프로필 | - PDF 생성(비동기, 15분 유틸링크)<br>- 기록 공유, 다운로드<br>- 당사자 자기기록 · 지원 만족도 조사 | SHARE API, PROFILE API |
| **W13**<br>09.04~09.16 | **FE L5~L6 구현** | - **L5**: ScreenVM, FormState<br>- **L6**: Layout, Component | - 공통 Layout(SideNav + TopBar + BottomBar)<br>- 반응형 브레이크포인트 세팅(S/M/T/D/L)<br>- Auth · Family · Record · Search 화면<br>- 템플릿 편집기 | FE Auth Page, FE Family Page, FE Record Page |
