## 1. 역할별 워크플로우 (Persona Workflows)

### 1.1 P1 : Guardians — Guardian Workflow

#### Goal
기관 전환 시 필요한 모든 서류를 플랫폼에서 생성하고, 보호자와 당사자가 기록을 함께 관리

```
[Start]
  │
  ├── ① 회원가입 (이메일 + 역할: 보호자 선택)
  │     │
  │     └── 이메일 인증 완료
  │
  ├── ② 당사자 추가
  │     │
  │     ├── 기존 당사자 연결 (기존 계정) 또는 신규 등록
  │     └── 당사자 정보 입력: 이름, 생년월일, 장애등급, 생애주기
  │
  ├── ③ 기록자 초대
  │     │
  │     ├── 사회복지사 초대: 이메일 → 역할 지정 → 접근범위 지정
  │     ├── 특수교사 초대: 이메일 → 역할 지정 → 접근범위 지정
  │     ├── 활동지원사 초대: 이메일 → 역할 지정 → 기간 지정
  │     └── 초대 상태 추적: "대기중" → "수락완료" / "만료"
  │
  ├── ④ 기록 작성 / 확인
  │     │
  │     ├── 직접 생성: 생애주기별 템플릿 선택 → 작성 → 저장/출력
  │     ├── 협업: 전문인력 작성 → 확인 → 수정요청 / 승인
  │     └── 열람: 모든 기록자 작성 기록 확인
  │
  ├── ⑤ PDF 생성 / 다운로드
  │     │
  │     ├── 기록 선택 (단일 / 복수 선택)
  │     ├── PDF 생성 대기 (비동기)
  │     └── 생성 완료 → 다운로드 / 공유
  │
  └── ⑥ 권한 관리
        │
        ├── 기록 공개범위 설정: 나만보기 / 제한공개 / 공유
        ├── 기록자 권한수정 / 회수
        └── 당사자 만 18세 → 권한 이양 알림 확인
```

### 1.2 P2 : 당사자 Adult — 당사자 Adult Workflow

```
[Start] 성인 당사자
  │
  ├── ① 계정 생성 (이메일 + 역할: 당사자)
  │     │
  │     └── 자동 로그인 (만 18세 이상 판단)
  │
  ├── ② Easy Mode 진입
  │     │
  │     └── Large Text, Large Button, Simple Icon
  │
  ├── ③ 기록 열람
  │     │
  │     ├── 최근 기록 보기 (카드 뷰)
  │     ├── 생애주기 타임라인 확인
  │     └── 기록자별 작성 기록 필터
  │
  ├── ④ 직접 기록 작성
  │     │
  │     ├── 자기소개서 작성
  │     ├── 선호 / 관심사 기록
  │     └── 지원 서비스 만족도 조사
  │
  └── ⑤ 알림 확인
        │
        └── 새로운 기록 added, 예정된 일정, 초대수락요청
```

### 1.3 P3 : 사회복지사 — Social Worker Workflow

```
[Start] 사회복지사
  │
  ├── ① 기관 코드 인증
  │     │
  │     ├── 소속 기관명 검색
  │     ├── 기관 코드 요청 (관리자 승인)
  │     └── 인증 완료
  │
  ├── ② 케이스 관리
  │     │
  │     └── 담당 케이스 목록: 이름, 생애주기, 기록수, 마지막활동
  │
  ├── ③ 기록 작성 (Template Based)
  │     │
  │     ├── 템플릿 선택: 생애주기 + 카테고리
  │     ├── 동적 폼 렌더링 (10종 필드 타입)
  │     ├── 파일을 첨부
  │     └── 저장 (임시 / 확정)
  │
  ├── ④ 기록자 협업
  │     │
  │     ├── 보호자 초대 요청
  │     ├── 특수교사 / 활동지원사 공유
  │     └── 기록 수정 이력 추적
  │
  └── ⑤ 서비스 연계
        │
        └── 현재 서비스 목록 → 다음 서비스 계획 → 기록
```

### 1.4 P4 : 특수교사 — Special Teacher Workflow

```
[Start] 특수교사
  │
  ├── ① 학교 (인정)
  │     │
  │     └── 학교 이름 및 기관코드 검색
  │
  ├── ② IEP 작성
  │     │
  │     ├── IEP 템플릿 선택 (생애주기별 상동)
  │     ├── 목표설정 (장기 / 단기)
  │     ├── 평가작성 (수행 / 개선사항)
  │     └── 보호자 / 사회복지사와 공유
  │
  └── ③ 졸업 후 기록 관리
        │
        └── 졸업생: 일정 기간 조회 가능 → 이후 열람 전용
```

### 1.5 P5 : 활동지원사 — Support Worker Workflow

```
[Start] 활동지원사
  │
  ├── ① 로그인 (이메일 + 비밀번호)
  │
  ├── ② 오늘의 지원일지 작성
  │     │
  │     ├── 지원 대상자 선택
  │     ├── 지원 내용 작성: 이동 / 식사 / 개인위생 / 취미
  │     ├── 특이사항 기록: 사고 / 긴급상황
  │     └── 사진 첨부 (지원 활동 증거)
  │
  └── ③ 주간 리포트
        │
        └── 작성된 일지 자동 집계 → 주간 요약 확인
```

## 2. 생애주기별 워크플로우 (Lifecycle Workflows)

### 2.1 생애주기 정의

| Phase | Age Range | Life Stage | Description |
|-------|-----------|------------|-------------|
| **Infancy** | 0-5 세 | 영유아 | 건강검진, 발달검진, 조기교육 |
| **Early Childhood** | 6-11 세 | 유아 / 초등 | 특수교육 IEP, 생활기록 |
| **Adolescence** | 12-17 세 | 청소년 | 진로상담, 자립준비, 사회기술 |
| **Young Adult** | 18-29 세 | 청년 | 직업훈련, 자립지원, 사회참여 |
| **Middle Adult** | 30-59 세 | 성인 | 직업재활, 지역사회활동, 가정치료 |
| **Older Adult** | 60세+ | 노년 | 건강관리, 돌봄서비스, 여가·문화 |

### 2.2 생애주기별 기록 워크플로우

#### Infancy (0-5 세)

```
[보호자] 등록
  │
  ├── ① 출생등록
  │     └── 이름, 생년월일, 성별, 진단정보
  │
  ├── ② 발달검진
  │     ├── K-DST, PEDI, GMs 등
  │     └── 매 6개월 간격 권장
  │
  └── ③ 조기교육
        ├── 조기교육 시설 등록
        ├── 프로그램 참여 기록
        └── 지도교사 기록 → 특수교사 협업
```

#### Early Childhood (6-11 세)

```
[특수교사] IEP 수립
  │
  ├── ① IEP 목표설정
  │     ├── 장기목표 (1년)
  │     ├── 단기목표 (학기별)
  │     └── 평가기준 정의
  │
  ├── ② 실행 및 기록
  │     ├── 학업 성취도 기록
  │     ├── 사회기술 기록
  │     └── 행동 관찰 기록
  │
  ├── ③ 보호자 공유
  │     └── 가정에서 이어지는 지원 기록
  │
  └── ④ 졸업전환
        └── 중고등학교 전환시 기록 이전
```

#### Adolescence (12-17 세)

```
[보호자] + [당사자] 협업
  │
  ├── ① 진로 탐색
  │     ├── 관심사 조사 기록
  │     ├── 진로 상담 기록
  │     └── 체험 활동 기록
  │
  ├── ② 자립준비
  │     ├── 일상생활기술 훈련
  │     ├── 금융교육
  │     └── 주거 훈련
  │
  ├── ③ 사회기술 향상
  │     ├── 대인관계 기록
  │     ├── 감정 조절 기록
  │     └── 위기 대처 기록
  │
  └── ④ 만 18세 준비
        ├── 권리가양 안내 (D-90 / D-30 / D-7)
        ├── 후견인 범위 파악
        └── 이양 준비 대화 기록
```

#### Young Adult (18-29 세) — 권리가양 시점

```
[시스템] 자동 전환
  │
  ├── ① 만 18세 알림 (D-90 / D-30 / D-7)
  │     │
  │     ├── 보호자: 이양안내 + 설명
  │     ├── 당사자: 자기결정권 안내
  │     └── 이양 프로세스 시작
  │
  ├── ② 전양 프로세스
  │     │
  │     ├── 전자동의서 서명
  │     ├── PASS 본인인증
  │     ├── 보호자 쓰기 권한 → 열람 전용 전환
  │     └── 당사자에게 모든 권리 부여
  │
  ├── ③ 당사자 주도의 기록 관리
  │     ├── 당사자가 직접 기록 작성
  │     ├── 기록자 관리 (초대 / 권한수정 / 회수)
  │     └── 기록 공개범위 설정
  │
  └── ④ 직업 / 사회참여
        ├── 직업훈련 기록
        ├── 직장 적응 기록
        └── 지역사회 활동 기록
```

#### Middle Adult (30-59 세)

```
[당사자] 주 활동
  │
  ├── ① 직업재활
  │     ├── 직종 기록
  │     ├── 성과 기록
  │     └── 퇴사 / 이직 기록
  │
  ├── ② 지역사회 활동
  │     ├── 지역사회 자원 연결 기록
  │     ├── 그룹활동 참가
  │     └── 사회참여 기록
  │
  └── ③ 건강관리
        └── 정기검진, 치료, 약 기록
```

#### Older Adult (60세+)

```
[당사자] + [보호자] 협업
  │
  ├── ① 돌봄 서비스
  │     ├── 일상생활 지원 기록
  │     ├── 방문간호 기록
  │     └── 요양 서비스 기록
  │
  ├── ② 여가·문화
  │     ├── 취미 활동
  │     ├── 문화 참여
  │     └── 사회관계 네트워크
  │
  └── ③ 마지막 삶의 기록
        ├── 유언·장례 사전협의
        ├── 재산관리 기록
        └── 가족 메시지
```

## 3. 업무별 프로세스 (Business Process Flows)

### 3.1 B001 : 템플릿 기반 기록 작성

```
[시작]
  │
  ├── ① 템플릿 선택
  │     ├── 생애주기 선택 (6단계) + 카테고리 선택
  │     └── 25개 템플릿 중 1개 선택
  │
  ├── ② 동적 폼 렌더링
  │     ├── 필드 타입: 텍스트, 숫자, 선택, 날짜, 파일, 체크박스
  │     └── 조건부 필드: 이전 답변에 따라 다음 필드 렌더링
  │
  ├── ③ 작성
  │     ├── 필수 필드 검증
  │     ├── 파일 첨부 (10MB max, MIME + Magic Bytes)
  │     └── 임시저장 가능
  │
  ├── ④ 저장
  │     ├── 확정 저장 (버전 이력 생성)
  │     └── 공개범위 설정
  │
  └── [끝]
```

### 3.2 B02 : 기록자 관리

```
[시작] — 기록자 초대
  │
  ├── ① 초대 요청
  │     ├── 대상자 이메일 입력
  │     ├── 역할 지정 (사회복지사/특수교사/활동지원사/보호자)
  │     ├── 접근범위 지정 (카테고리)
  │     └── 기간 지정 (선택)
  │
  ├── ② 이메일 발송
  │     ├── 초대 링크 (7일 유효)
  │     └── 상태: "대기중"
  │
  ├── ③ 수락 / 거절
  │     ├── 수락 → 계정 생성 → 권한 부여
  │     └── 거절 / 만료 → "초대취소"
  │
  ├── ④ 권한수정
  │     ├── 접근범위 변경
  │     └── 종료일 변경
  │
  └── ⑤ 권한 회수
        ├── 즉시 적용
        ├── 이메일 알림 발송
        └── 권한 이력 기록
```

### 3.3 B003 : PDF 생성 및 출력

```
[시작]
  │
  ├── ① 기록 선택
  │     ├── 단일 기록
  │     ├── 날짜 범위 내 전체
  │     └── 카테고리별 전체
  │
```

## 4. 권한 이양 워크플로우 (Permission Transfer Flow)

### 4.1 상태 전도

```
[NEW_BORN]
  │
  ├── 전환 조건: 만 18세 생일 D-90
  │   └── 상태: TRANSITION_PLANNED
  │
  ├── 전환 조건: 만 18세 생일 D-30
  │   └── 상태: TRANSITION_REMINDER
  │
  ├── 전환 조건: 만 18세 생일 D-7
  │   └── 상태: TRANSITION_NOTIFIED
  │
  ├── 전환 조건: 당사자 이양 시작 버튼 클릭
  │   └── 상태: TRANSITION_IN_PROGRESS
  │
  ├── 전자서명 완료
  │   └── 상태: TRANSITION_VERIFIED
  │
  ├── PASS 인증 완료
  │   └── 상태: TRANSITION_AUTHENTICATED
  │
  ├── 보호자 쓰기 권한 → 열람 전용 전환
  │   └── 상태: TRANSITION_COMPLETE
  │
  └── 당사자에게 모든 권한 부여
```

### 4.2 예외 처리

| Case | Handling |
|------|----------|
| **이양 거부** | TRANSITION_REJECTED 상태로 전환. 보호자 다시 쓰기 권한 유지 |
| **법정후견인 있음** | MANAGE_OVERRIDE 등록. 후견범위 내에서만 후견인이 권한 |
| **복수보호자** | 모든 보호자의 동의를 받아야 함 |
| **탈주** | 상태 유지. 30일 후 다음 주기 알림 재발송 |
